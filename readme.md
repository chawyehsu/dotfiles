<h1 align="center">Dotfiles</h1>

yet a simple, uniform terminal style and dotfiles, for macOS, Windows, and Linux.

Usage
-----

``` powershell
$ git clone https://github.com/chawyehsu/dotfiles ~/.dotfiles
$ cd ~/.dotfiles && ./install.ps1
...
```
